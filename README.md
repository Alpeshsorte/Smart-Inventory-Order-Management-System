A modern Inventory & Order Management System built using **HTML, CSS, JavaScript, and JSON Server**.

This project simulates a real-world e-commerce admin dashboard used in electronics stores and warehouses for managing products, orders, stock, search, filtering, sorting, and pagination.

---

# Features

## Product Management

* Add Product
* Update Product
* Delete Product
* View Products
* Product Card Design

## Search Functionality

Search products by:

* Product Name
* Brand
* Category

## Filter Functionality

Filter products using:

* Category
* Brand
* Stock Status
* Price Range

## Sorting Functionality

Sort products by:

* Price Low to High
* Price High to Low
* Product Name A-Z
* Product Name Z-A

## Pagination

* Next Page
* Previous Page
* Dynamic Page Buttons
* Active Page Highlight

## Dashboard Analytics

* Total Products
* Total Orders
* Total Revenue
* Available Stock
* Pending Orders

# JSON Server Setup

## Install JSON Server

```bash
npm install -g json-server
```

---

## Run JSON Server

```bash
json-server --watch db.json
```

Server runs on:

```bash
http://localhost:3000
```

# API Endpoints

## Products

```bash
GET /products
POST /products
PUT /products/:id
DELETE /products/:id
```

## Orders

```bash
GET /orders
POST /orders
PUT /orders/:id
DELETE /orders/:id
```
